Ejercicio 10.7 | Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. Para hacer referencia a una ubicación o elemento específico en un arreglo, especificamos el nombre del arreglo y el valor del elemento.

    2. > Verdadero.

  2. La declaración de una variable reserva espacio para un arreglo.

    3. > Verdadero.

  3. Para indicar que deben reservarse 100 ubicaciones para el arreglo entero p, el programador debe escribir la siguiente declaración: p: [100];

    4. > Falso, la sintaxis es incorrecta.

  4. Un programa en JavaScript que inicializa los elementos de una arreglo de 15 elementos en cero debe contener cuando menos una instrucción for.

    5. > Falso, dependiendo de la función que tendra el arreglo será la forma en como se use con un bucle.

  5. Un programa de JavaScript que calcula el total de los elementos de un arreglo bidimensional debe contener instracciones for anidadas.

    6. > Verdadero.
