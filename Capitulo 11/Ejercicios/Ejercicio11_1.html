Inicio

Ejercicio 11.1 | Completa las siguientes oraciones:

  1. Como JavaScript usa objetos para realizar muchas tareas, se conoce por lo común como __lenguaje de programación basado en objetos__.

  2. Todos los objetos tienen __atributos__ y exhiben __comportamientos__.

  3. Los métodos del objeto __Math__. nos permiten realizar muchos cálculos matemáticos comunes.

  4. Al proceso de invocar (o llamar) al método de un objeto se le conoce como __enviar un mensaje al objeto__.

  5. Las literales de cadena o constantes de cadena se escriben como una secuencia de caracteres en __comillas dobles__ o __comillas sencillas__.

  6. Los índices para los caracteres en una cadena empiezan en __0__.

  7. Los métodos __indexOf__ y __lastIndexOf__ de String buscan la primera y última ocurrencias de una subcadena en un objeto String, respectivamente.

  8. Al proceso de dividir una cadena en tokens se le denomina __tokenización__.

  9. El procesamiento de fechas y horas puede realizarse con base en la __zona horaria local__ o en el __tiempo universal coordinado__ del Estándar de Tiempo Mundial.

  10. El método __parse__ de Date recibe como argumento una cadena que representa una fecha y hora, y devuelve el número de milisegundos transcurridos entre la medianoche del 1 de enero de 1970 y la fecha y hora especificadas.

  11. Las aplicaciones Web pueden usar la propiedad __localStorge__ del objeto window para almacenar hasta varios megabytes de datos de cadenas de pares clave/valor en la computadora del usuario y pueden acceder a estos datos a través de sesiones y pestañas del navegador.

  12. Las aplicaciones Web que necesitan acceso a los datos de pares clave/valor para sólo una sesión de navegación y que deben mantener esos datos separados entre varias pestañas pueden usar la propiedad __sessionStorge__ del objeto window.

  13. Un selector __ :first-child __ de CSS3 selecciona el primer hijo de un elemento.

  14. Un selector __nth-child__ de CSS3 con el argumento "odd" selecciona los elementos hijos impares y uno con el argumento "even" selecciona los elementos hijos pares.
