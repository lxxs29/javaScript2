Ejercicio 12.1 | Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. Cada elemento de HTML5 en una página se representa mediante un árbol DOM.

    >>> Falso, todo elemento se representa mediante un nodo DOM.

  2. Un nodo de texto no puede tener nodos hijos.

    >>> Verdadero

  3. Es posible cambiar la clase de estilo de un elemento en forma dinámica, estableciendo el atributo style.

    >>> Falso, el documento es el nodo raíz, por lo tanto no puede tener un nodo padre.

  4. El método createElement crea un nuevo nodo y lo inserta en el documento.

    >>> Falso, la clase de estilos se modifica estableciendo el atributo class.

  5. El método setInterval llama a una función repetidas veces, con un intervalo de tiempo establecido.

    >>> Falso, el método createElement crea un nodo pero no lo inserta en el árbol DOM.

  6. El método insertBefore se llama en el objto documento; recibe un nuevo nodo y uno existente para insertar el nuevo antes.

    >>> Verdadero

  7. El intervalo más reciente iniciado se detiene al momento de llamar al método clearInterval.

    >>> Falso, clearInterval recibe un identificador de intervalo como argumento oara determinar qué intervalo terminar.

  8. La colección links contiene todos los vínculos en un documento con un atributo id especificado.

    >>> Falso, la colección links contiene todos los vínculos del documento.
