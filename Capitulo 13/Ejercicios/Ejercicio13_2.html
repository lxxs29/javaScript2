Ejercicio 13.2 | Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. El evento load se dispara cada vez que un elemento empieza a cargarse.

    >>> Falso, el evento load se dispara cuando el elemento termina de cargarse.

  2. El evento click se dispara cuando el usuario hace clic con el ratón en un elemento.

    >>> Verdadero

  3. El evento focus se dispara cuando un elemento pierde el enfoque.

    >>> Falso, se dispara cuando un elementoo gana el enfoque.

  4. Al usar el efecto de sustitución con imágenes, es una buena práctica crear objetos image que realicen cargas previas de las imágenes deseadas.

    >>> Verdadero

  5. Al regresar true en un manejador de eventos en un elemento a (ancla) se evita que el navegador siga el vínculo cuando termine el manejador de eventos.

    >>> Falso, al devolver false se evita la acción predeterminada.
